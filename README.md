# TwistedPlant - Premium Streetwear

Welcome to **TwistedPlant**, a premium streetwear brand crafted for those who refuse to blend in. Every thread tells a story of rebellion, quality, and uncompromising style.

> **"Wear Your Identity"**

## Project Overview

TwistedPlant is a high-quality Print-On-Demand e-commerce storefront. It features a modern, responsive design with a focus on visual storytelling and seamless user experience.

### Key Features

-   **Premium Aesthetics**: Sleek, modern tailored design with smooth animations.
-   **Showcase Ready**: Dedicated sections for specific product collections and brand storytelling.
-   **Responsive Layout**: Fully optimized for mobile, tablet, and desktop.
-   **Tech Stack**: Built with the latest web technologies for speed and maintainability.

## Tech Stack

This project is built with:

-   **[Vite](https://vitejs.dev/)**: Next-generation frontend tooling.
-   **[React](https://reactjs.org/)**: Library for building user interfaces.
-   **[TypeScript](https://www.typescriptlang.org/)**: Typed JavaScript for better tooling and error prevention.
-   **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework for rapid UI development.
-   **[shadcn/ui](https://ui.shadcn.com/)**: Reusable components built with Radix UI and Tailwind CSS.
-   **[TanStack Query](https://tanstack.com/query/latest)**: Powerful asynchronous state management.

## Getting Started

Follow these steps to run the project locally:

### Prerequisites

-   Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation & Run

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_DIRECTORY>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Deployment

You can deploy this project to any static hosting service like Vercel, Netlify, or GitHub Pages.

```sh
npm run build
```

## Customization

To customize the content:
-   **Products**: Edit `src/components/ProductShowcase.tsx`
-   **Story**: Edit `src/components/BrandStory.tsx`
-   **Navigation**: Edit `src/components/Header.tsx` and `src/components/Footer.tsx`
